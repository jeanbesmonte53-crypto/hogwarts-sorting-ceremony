# Hogwarts Sorting Ceremony

## Overview
The Hogwarts Sorting Ceremony web application is designed to simulate the magical sorting process that students undergo when they arrive at Hogwarts School of Witchcraft and Wizardry. This application provides a fun and interactive way for users to experience the sorting process and learn about different Hogwarts houses.

## Features
- **Interactive Sorting Quiz**: Users can answer a series of questions to determine their house.
- **House Profiles**: Detailed information about each house, including values, celebrities, and more.
- **User Accounts**: Users can create and manage accounts to save their sorting results.
- **Mobile-Friendly**: The web application is responsive and works on all devices.

## Quick Start Guide
1. Clone the repository:  
   ```bash
   git clone https://github.com/jeanbesmonte53-crypto/hogwarts-sorting-ceremony.git
   ```  
2. Navigate to the project directory:  
   ```bash
   cd hogwarts-sorting-ceremony
   ```  
3. Install dependencies:  
   ```bash
   npm install
   ```  
4. Start the application:  
   ```bash
   npm start
   ```  
5. Open your browser and navigate to `http://localhost:3000`

## Configuration
- **Environment Variables**: 
  - `DATABASE_URL`: URL for the database connection.
  - `JWT_SECRET`: Secret for JWT authentication.
- Configuration files can be found in the `/config` directory.

## Troubleshooting
- **Application not starting**: Ensure all dependencies are installed and try running `npm install` again.
- **Database connection issues**: Verify your database URL and make sure the database server is running.
- **Common Errors**: Check the logs for any error messages that can guide you toward a solution.
